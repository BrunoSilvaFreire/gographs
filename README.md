# Gographs

A Go library for graph algorithms and data structures.

## Features

- Adjacency List and Matrix implementations.
- Topological Sort (Kahn's algorithm).
- Cycle Detection.
- BFS and DFS Traversals.
- A* Search algorithm.

## Installation

```bash
go get github.com/BrunoSilvaFreire/gographs
```

## Usage

```go
import "github.com/BrunoSilvaFreire/gographs/pkg"
import "github.com/BrunoSilvaFreire/gographs/pkg/algorithms"

// ...
```
